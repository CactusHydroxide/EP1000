What is 3D modelling?

3D modelling is the process of using vertices, edges and surfaces to create a 3D model. It requires the use of specialised applications to be done. 3D modelling serves many purposes. Models can be used in art, animation, games and even printed. We will be 3D modelling with the intent to print our models.

What is 3D printing?

Usually referring to the action of depositing a material layer by layer until you have a finished product, 3D printing is a form of additive manufacturing that constructs a 3D object from a CAD model. It is commonly used for rapid prototyping and creating small objects.

How to create a 3D model?

To learn how to 3D model, we will be making a chess knight piece. This knight piece will be 3D printed later.

Part 1 - Setup

Step 1

Create rectangle with 2-point rectangle.

Step 3

Use midpoint tool to select the bottom of the rectangle, and click the origin point to centre the rectangle. This helps to center the rectangle to the origin.

Step 4

Use sketch dimension to define the height and width of the rectangle. H:50 W:30. This helps to create a reference for how large our model will be. When you are done it should look something like this.

Step 5

Look for a reference image. Using a reference image will make the process of modelling significantly easier. If you are unable to find a reference image online, you can always draw it out on paper or using any illustrative tool. Once you have a reference image, we will insert it into our canvas. To do so, go to . Insert > Canvas > Insert from my computer, and search for your image. To insert canvas, click the previously created rectangle. Flip and scale the reference image accordingly.

Step 6

Press the finish sketch button on top and hide rectangle. The rectangle is only used to scale the horse. Finally, hide the sketch of the rectangle using the tree on the right.

Part 2 - Base

Step 1

Create a new sketch and select a new plane to model the horse.

Step 2

Under the create tab, use the line, arc and spline tool to out line half of the knights base. Draw out half of the knight's base.

Step 3

Select the base sketch. Press revolve. In the menu, select axis set the point of rotation to the y-axis. Next, set the type to full. Lastly ensure that the operation creates a new body. Once you are done click OK. Your base of the chest piece is complete.

Part 3 - Horse

Step 1

Create new sketch and draw out the horse. It follows the same steps as sketching out the base. Take your time as you go along the curves of the knight piece to get all the details. Leave the mane out as we will be doing it separately.

Step 2

Once you have finished the outline of the horse, select the extrude function. Follow the settings as shown bellow. Take note of which measurement you choose. If you pick the first one, your distance refers to the distance between the center and the side. If you choose the second one, the distance is the thickness of the entire structure. Lastly, remember to set the operation to join as this top piece is going to attach to the base.

Step 3

Next, create a new sketch on the surface of the ears. On that surface, draw out how you would like the ears to look. Lastly, use the extrude function to remove the excess. Remember to set the distance as a negative number and set the operation to cut.

Step 4

Lastly remove the excess on the top of the horses ear/

Step 1

Create a new sketch of the mane of the horse as previously done.

Step 2

Using the extrude function, do the same as before. Do remember to keep the distance thinner than the body of the horse, and set the operation back to join.

Next we will chamfer the edges to make the horse look better. Select all the edges that you would like to chamfer.

Parametric Modelling

What is parametric modelling?

Parametric modelling is 3D modelling with the use of pre-programmed measurements and algorithms as measurements and functions. These rules and algorithms are known as parameters. You can think of them as variable in code or letters in algebra. They are used to reference a specific value or action.

Why use parametric modelling?

Parametric modelling allows one to quickly change the parameters on the fly. By changing the parameters, values and measurements can be changed across the entire model as long as they are properly referenced. For example, if you created a model of a box but you want the width to be longer. In direct modelling you are required to change the width for every time the value is used in the model. However, when utilising parametric models, as long as you have referenced your parameters, you can change the parameters to change the values of all 'widths' at the same time.

How to make a parametric model?

Before you start modelling, you are first required to set up parameters. To do so select change parameters under modify. (Make sure you are not in a sketch first else you cannot find it).

To add parameters, click on the plus button beside user parameters. Next, add a name, unit (if applicable) and expression into the parameter. The expression could be a single value or a complex algorithm to find any value. The value tab should fill up automatically. Comments are optional but it is highly advised that you do so to remember what the parameters are used for.

This is the parameters I used for my music box. The box will show up in a few places in this website so keep your eyes peeled!

Lastly, ensure that ALL measurements when modelling, reference the parameters instead of a fixed value. Created a fixed valued leads to the problem resizing different parts of the box will not work as intended later.

Take note that my height, width and thickness are all parameters instead of values.

You can also use parameters as the measurements of guides. To do so, just set the measurements of your guide lines to the name of the parameter instead of using a defined value.

As you can see, both the square and thickness of the circle are using parameters to help define its value. This allows me to resize them later depending on the sizing of my box.
