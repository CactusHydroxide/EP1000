Tools

Here are the tools I used for my projects.

Application Usage Notes
Visual Studio Code Rich text editor used for code. Mainly used for website development Free software, customisable, already used for other projects.
Adobe Photoshop Raster image editor Industry standard software. Paid, but I have it for free from school.
Adobe Illustrator Vector image editor Industry standard software. Paid, but I have it for free from school.
Snip & Sketch Screen capture tool Quick screen capture application. Comes pre-installed on Windows.
Fusion 360 Computer-assisted design (CAD) software Free software. Recommended by lecturer.
TinkerCAD - Circuits Online Arduino testing software Online freeware that allows me to test my electronics and code without blowing LEDs or any parts.
Arduino IDE Integrated Development Environment Designed specially to be used with Arduino.
