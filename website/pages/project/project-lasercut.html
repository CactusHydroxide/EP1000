What is Laser Cutting?

Laser Cutting is the process of using a high power laser with pin point precision to cut sheets of material. It allowed you to be able to cut sheets of material very precisely. It also allows you to be able to engrave in the material. This gives you a lot of flexibility on your design.

Laser Cutting - Designing

The laser cutting machine requires a Vector image file to know what it is going to cut. The colours of the graphic refers to different laser cutting settings. For this project we will be using 2 colours, red and black. Red will refer to cutting and black will refer to engraving.

I will be using Adobe Illustrator to create my vector file. The design we will be making is that of a keychain with a QR code. Before we start, make sure you have a reference image ready for your QR code, or any design you would like to create.

Step 1

First we have to set up our AI file. When creating a new file, ensure that you create it with the dimensions of 10 x 10 cm. Colour mode and raster effects would not affect the file so do not worry too much about them. However, most would prefer to set Colour Mode to RGB.

Step 2

Next, insert your reference image into your file. Adjust the size of the reference image to 3.5cm x 3.5cm Once you have done so, center it on the page and lock it on the layer window. (Use F7 to bring up the layers window)

Step 3

Now, we need to convert, this raster image to a vector graphic. However, due to the nature of the QR code, we do not need to convert it to a vector. If you are using another raster image, do look at Vector Graphic to learn more.

Step 4

Next we will be draw the outline of the keychain. Use the curve rectangle tool to draw a 4cm x 4cm square. You can left click while selecting the tool to choose the exact measurements of the rectangle.

Step 5

Once you have the outline, use the colour selection at the bottom to change the colour of the rectangle. Set the outline of the rectangle to red and remove the fill of the rectangle. Lastly, center the rectangle on the artboard.

Step 6

Now create 2 circles. They should have width and height of 1cm and 0.5cm respectively. Center the smaller circle inside the larger one and overlap them with the top of your keychain. It should look as follows.

Step 7

We will now be merging all the shapes together. Select only the outer circle and the rectangle and use the unite tool to merge both shapes. You can see which shape mode I used on the top left. If you do not see the pathfinder tab, press Ctrl + Shift + F9 to open it.

Step 8

Next, select the smaller circle and the outer shape. Then use the minus front tool to cut out the smaller circle from the main shape. Again, you can see the tool used on the top right

Step 9

Lastly, select the newly merged shapes and set the thickness to 0.25px. This will ensure that the cut will be as thin as possible.

Step 10

To make the engraving nicer, we would want to engrave on the back of the acrylic sheet. Select everything, right click > transform > reflect vertically. This will flip the design and prepare it for cutting.

Step 11

Ensure that you save your file as a vector. Save it as a AI, SVG or PDF. Ensure that your laser cutting is able to take your vector file format as an input.
